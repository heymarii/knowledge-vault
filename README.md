# 🗄️ Knowledge Vault

> A structured, searchable knowledge base. Unlike the second-brain (which is daily notes), this is curated, organized reference material — the kind of thing I want to find fast when I need it.

## What's Different From Second Brain?

| Second Brain | Knowledge Vault |
|-------------|-----------------|
| Raw daily notes | Polished reference docs |
| Chronological | Topically organized |
| "I just learned this" | "This is my definitive guide to X" |
| Updated daily | Updated when I refine my understanding |

## Structure

```
knowledge-vault/
├── concepts/        ← Explanations of important concepts
├── tools/           ← Guides to tools I use regularly  
├── frameworks/      ← Mental models and decision frameworks
├── cheatsheets/     ← Quick reference cards
└── decisions/       ← Decision logs (what I chose and why)
```

## How to Use

- **Search by topic:** Browse the folders or use GitHub's search
- **Cheatsheets:** Quick reference — meant to be opened, scanned, closed
- **Concepts:** Deeper explanations written in my own words
- **Decisions:** A record of important choices I've made and my reasoning
