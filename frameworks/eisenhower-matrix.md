# The Eisenhower Matrix

**Use for:** Prioritizing tasks when everything feels urgent

---

## The Framework

```
                    URGENT          NOT URGENT
              ┌──────────────────┬──────────────────┐
    IMPORTANT │   DO NOW         │   SCHEDULE       │
              │ (Quadrant 1)     │ (Quadrant 2)     │
              ├──────────────────┼──────────────────┤
NOT IMPORTANT │   DELEGATE       │   ELIMINATE      │
              │ (Quadrant 3)     │ (Quadrant 4)     │
              └──────────────────┴──────────────────┘
```

## The Four Quadrants

**Q1 — Do Now:** Crisis, deadline, emergency. These get done immediately.  
*Examples: broken production system, overdue deliverable, urgent client issue*

**Q2 — Schedule:** Important but not on fire. This is where leverage lives.  
*Examples: learning new skills, long-term projects, relationship building, health*

**Q3 — Delegate:** Urgent but not important to YOU. Someone else should handle it.  
*Examples: many meetings, some emails, interruptions*

**Q4 — Eliminate:** Neither urgent nor important. Just stop doing these.  
*Examples: mindless browsing, busywork, low-value tasks that persist by habit*

## The Key Insight

Most people spend time in Q1 (fighting fires) and Q4 (low-value easy tasks). High performers protect Q2 time — that's where growth happens.

## How I Use This

1. Each Sunday, list everything I think I need to do this week
2. Put each item in a quadrant
3. Calendar-block Q2 items first (they never happen otherwise)
4. Q1 items handle themselves (they're on fire)
5. Ask "can I delete this?" for every Q4 item

---

*Source: From Eisenhower's approach to time management, popularized by Stephen Covey in "The 7 Habits of Highly Effective People"*
